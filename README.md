# Clothing Store Backend API

Backend system for a clothing store application built using Laravel and MySQL.

## Features

* User authentication and authorization
* Product management system
* Product variants handling (size & color)
* Shopping cart logic
* Order management
* Inventory and stock quantity control
* RESTful APIs for mobile application
* Admin dashboard functionality

## Technologies Used

* Laravel
* PHP
* MySQL
* REST API
* Git & GitHub

## Database Design

Implemented relational database structure including:

* products
* product_variants
* colors
* sizes
* carts
* orders
